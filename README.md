## 📦 R Packages & Software

### [`refreg`](https://cran.r-project.org/package=refreg) — *CRAN*
> Conditional reference region estimation via copula-based additive models. Computes multivariate tolerance and prediction regions for clinical laboratory data. Published in [**The R Journal**](https://journal.r-project.org/).

```r
install.packages("refreg")
```

---

### [`TwoDiRef`](https://github.com/Bioscar/TwoDiRef) — *GitHub*
> Conditional **2D robust reference regions** using median-based partial correlation models and directional quantile regression. Includes k-fold cross-validation for empirical coverage and an interactive **Shiny app**.

```r
remotes::install_github("Bioscar/TwoDiRef")
```

---

### [`mctmref`](https://github.com/Bioscar/mctmref) — *GitHub*
> Reference regions based on **multivariate conditional transformation models** (MCTM). A flexible framework for estimating multivariate reference and tolerance regions grounded in transformation model theory.

```r
remotes::install_github("Bioscar/mctmref")
```
