About

Biostatistician and statistical methodologist with over 8 years of experience in clinical and statistical research. Based at University of Oviedo, I specialised in flexible distributional regression models, functional data analysis, compositional data (CoDA), and R package development. My work spans multivariate regression, functional distributional data, glycomics, continuous glucose monitoring, endocrinology among others. Always bridging rigorous statistical methodology with clinical relevance.

I collaborate actively with international institutions in Denmark, Switzerland, Germany, Portugal, the Netherlands, and the USA, and I hold over 400 hours of university teaching in statistics across four Spanish universities.

Methodological Toolkit

Regression & Inference

  • Logistic regression (binary & multinomial) — OR, 95% CI
  • Cox proportional hazards — HR, censoring, stratification
  • GAM with penalised splines (non-linear effects)
  • GAMLSS with non Gaussian heteroscedastic responses
  • Multi-state models & semi-competing risks
  • Poisson regression with population offset (spatial)
  • Non-parametric tests: U Mann-Whitney, Kruskal-Wallis, Wilcoxon.
  • Multiple testing correction: Bonferroni, Benjamini-Hochberg FDR.

Variable Selection & Multivariate Statistics

  • Elastic Net (L1/L2) with C-index cross-validation
  • Regression Trees and Random Forest
  • Stepwise backward selection (likelihood ratio test)
  • PCA (classical & robust) — elbow method
  • Hierarchical clustering + K-means

Functional (Distributional) Data Analysis

  • Multilevel functional response regression (CGM, 5-min intervals)
  • Time-varying coefficients b(t)
  • Functional R² (marginal & joint)
  • Optimal Cutpoints for Functional Data (Digital Biomarkers)

Compositional Data Analysis (CoDA)

  • selbal algorithm: optimal balance search (forward)
  • Lasso regression on balances
  • Isometric log-ratio (ilr) transformation
  • EM log-ratio imputation for rounded zeros
  • Applications: glycomics, microbiome, % time in range

Model Validation

  • Bootstrap correction of classification optimism
  • ROC / AUC with DeLong CI
  • Calibration plots (observed vs. predicted) and Brier score
  • Harrell's C-index (survival models)
  • NRI & IDI (incremental value)
  • Nomograms & clinical scores (e.g., Gal-COVID-19, GASENT)

Missing Data & Survey Design

  • MICE. Several imputed datasets: Rubin's rules
  • EM imputation for compositional zeros
  • Imputation on ilr scale + back-transform to simplex
  • Sensitivity analysis: imputed vs. complete case
  • Compensatory weights (complex stratified sampling)
  • Design-based analysis for unequal selection probabilities

R Packages

{}
refreg

Conditional reference region estimation using flexible nonparametric and semiparametric models. Implements multivariate tolerance and prediction regions via copula-based additive models. Published in The R Journal.

→ CRAN · refreg
{}
TwoDiRef

Conditional 2D robust reference regions via median-based partial correlation models and directional quantile regression. Supports cross-validation coverage assessment and an interactive Shiny app for region exploration. With Sara Rodríguez-Pastoriza, J. Roca-Pardiñas & F. Gude.

→ GitHub · Bioscar/TwoDiRef
{}
mctmref

Reference regions based on multivariate conditional transformation models (MCTM). Provides a flexible framework for estimating multivariate reference and tolerance regions using transformation model theory.

→ GitHub · Bioscar/mctmref
